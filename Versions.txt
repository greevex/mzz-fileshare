	== Состав папки ==
	configs - конфигурация данного приложения
	files - папка для файлов, доступных извне, а так же для превью картинок.
	modules - вэб-директория модуля (если нужна, можно использовать системную директорию, но тогда файлы отдаютсячерез пхп, а не через вэб-сервер)
	nbproject - папка конфига проекта. руками не изменять. меняется сама черзе вэб-конфиг
	templates - папка с вэб-частью. темплейты, используемые smarty, а так же вэб-картинки, js, css и др. файлы
	tests - для тестирования. не использовать в проекте.
	== == ==  == == ==
